# Code of Conduct

## Our Commitment

We want DataSage to be a welcoming, respectful project for contributors of all
experience levels.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and ask clarifying questions.
- Focus on improving the project, not winning arguments.
- Be kind in code reviews and issue discussions.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling or intentionally disruptive behavior
- Publishing private information without consent

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict
participation when behavior violates this code of conduct.
