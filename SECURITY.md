# Security Policy

If you discover a security issue, please do not open a public issue with
sensitive details.

Instead, report it privately to the project maintainer or through your hosting
platform's private security reporting flow.

When reporting a vulnerability, include:

- A short description of the issue
- Steps to reproduce
- Potential impact
- Any suggested mitigation

This project is currently intended for local development and experimentation.
It should not be treated as production-ready without additional hardening.
